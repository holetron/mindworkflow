# Local Creative Flow

Local Creative Flow — инструмент для офлайн-планирования творческих проектов в виде mindmap-графа. Основной фокус — не автоматизация процессов как в n8n, а визуальная сборка всех артефактов: от текстов до медиа. Подробнее о запуске и структуре читайте в [README_LOCAL.md](README_LOCAL.md).
