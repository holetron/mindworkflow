# Local Creative Flow - Standalone Release

A powerful offline-first creative flow tool for AI-assisted content generation and mind mapping.

## Quick Start

1. Download the executable for your platform:
   - **Windows**: `local-creative-flow-windows.exe`
   - **Linux**: `local-creative-flow-linux`
   - **macOS**: `local-creative-flow-macos`

2. Run the executable directly - no installation required!

3. Open your browser to `http://localhost:4321`

4. Start creating nodes, connecting ideas, and generating content!

- You'll need to configure your own AI provider keys if you want AI functionality
