Features

Everything you need to build
AI workflows

A complete toolkit for designing, connecting, and executing intelligent workflows with a visual node-based editor.

🔗

Visual Node Editor

Build workflows by connecting nodes visually on an infinite canvas. Drag, drop, and wire up complex AI pipelines with ease.

🤖

Multi-AI Provider

Connect to OpenAI, Gemini, Replicate, Midjourney, Google AI Studio, and more. Switch providers without changing your workflow.

💾

Offline-First

Works without internet connection. All data is stored locally with SQLite. Your workflows and data stay on your machine.

⚡

Execution Engine

Run workflows with parallel and sequential execution modes. Monitor progress in real-time with detailed execution logs.

🎨

Image Generation

Create and process images with AI directly within your workflows. Support for multiple image generation providers.

🔧

Agent System

Automate workflows with intelligent agents that can reason, plan, and execute multi-step tasks autonomously.

Quick Start

Up and running in seconds

Clone, install, and start building AI workflows in just four commands.

terminal
# Clone the repository
$ git clone https://github.com/holetron/mindworkflow.git

# Navigate to the project
$ cd mindworkflow

# Install dependencies
$ npm install

# Start the development server
$ npm run dev
